Digital Magazine Publishing

Ask The alumni magazine was looking for a design more inline with its magazine-style. Content varied month to month so they wanted to make sure the site would always appear with a desirable layout and that issues content. A unique part of this website is previous content would only be available through the archive. The website would very much be a digital magazine.

Outcome We had struggled to manage the wireframing and design phase in previous projects. Our clients wanted to see mockups in all screen size for all the pages we designed. I provided leadership in creating a process to limit the amount of static comps to a few pages types. I would design reusable HTML components and styles that would be used to layout all the remaining pages. This process allowed for more iteration and allowed us to focus our time better. I was responsible for creating the sitemap, mobile-priority, wireframes, and working prototype. I produced all the front-end production code, much of which was recycled from the prototype.
