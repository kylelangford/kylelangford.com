Email Building Web App

Design and Generate HTML Emails

I proposed that we design a library of block patterns based on their existing email designs. We boiled the emails down to 10 or so components. I designed a UI for arranging the blocks and populating them with content. This app would render the HTML in a preview panel and also allow you to export the generated HTML.
