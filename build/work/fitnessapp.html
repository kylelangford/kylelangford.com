Elev8d Fitness

Interaction Design and Angular Development

Angular Video Player

Ask Elev8d Fitness was in trouble, they had a good idea but a terribly implemented web app. Elev8d provided users with a special type of guided yoga workout as well as full-length workout content. They needed a major refresh but at the same time needed a code rebuild. Many of the features they wanted to implemnt were deemed "impossible" due to how the code was originally written.

Outcome We were able to design a player that simplified the UI and provided the seamless playback experience that was expected. I provided technical leadership to design the functionality and UI of the video component. Jay's focus was intergrating branding elements and Christian was rebuilding the video API service. I rebuilt the angular video components and created transitions and animations to give a more native app feel.
