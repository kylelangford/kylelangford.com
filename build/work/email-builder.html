Drizly Email Builder

Web App

Feature Design

Ask Needed to display topics over a timeline...

I was coding https://drizly.com/ emails for a while and noticed that a lot of the work was repetitive. I proposed that we design a library of blocks based on their existing emails. I boiled the emails down to 10 or so components. I designed a UI for arranging the blocks and populating them with content. This app would render the HTML in a preview panel and also allow you to export the generated HTML.

Sketch of Idea

If you click the settings wheel next to a project on the left it will load. The page collapses, but if you click projects at the bottom right it will reopen the left sidebar. Try changing some of the content on the right it renders instantly. The accordions match with the blocks in the preview. You can drag and drop the order in the preview window. The form above the preview lets you add blocks. Feel free to play around.

Design of Idea

I like this project because I think this one has merit as a standalone product. The work to standardize their designs came out really nice. I also taught myself angularJS. If I had more time I would have made the right sidebar draggable, and made it so you could click and edit the preview directly.
