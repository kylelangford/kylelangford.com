Process allows for systemic and repeatable to approach to success.

Ask The alumni magazine was looking for a design more inline with its magazine-style. Content varied month to month so they wanted to make sure the site would always appear with a desirable layout and that issues content. A unique part of this website is previous content would only be available through the archive. The website would very much be a digital magazine.

Challenge We had struggled to manage the wireframing and design phase in previous projects for RWD. Our clients wanted to see mockups in all screen size for all the pages we designed. With this project I changed how we approached RWD designs as a company. I design a process in creating a process to limit the amount of static comps. I would design reusable HTML components and styles that would be used to layout all the pages not covered. This changed how we would approach building the sites and overall led to more efficent builds. This process also allowed for more iteration and time to be spent on design.

OutcomeI was responsible for creating the sitemap, mobile-priority, wireframes, and working prototype. I produced all the front-end production code, much of which was recycled from the prototype.
