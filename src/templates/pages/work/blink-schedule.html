New Feature Development

B'Link is an assement management website that is advertised as a "Simple, step-by-step suggestions for the best communications to use each month during your plan year." The site was well recieved but user struggled to know when to best use the digital communications. The client asked for a way to show samople communication plans. The client loved pdfs so that was the initial thought, we were able to convice them that the pdf would better be managed as a web content and that their drupal cms could hanlde creating a well designed schedule.

Schedule Needed a way to display topics over a timeline...

Initial Sketch

We needed a way to organize the content but be easy to update in the future. My initial Drawing looks a bit like a gant chart, but shows a layout that takes advantage of Drupal page building. I know we could use repeating fieldgroups to set arbitrary groups of topics.

Using the site taxonomy and a few preprosecor hooks we are able to organize the topics and sort them visually using flexbox.
 
$rowCount = count($rows);

// If 2
if ($rowCount == 2) {
  // If "Get the most out of your benefits" and "Take care of your health"
  if (array_key_exists("Get the most out of your benefits", $rows) && array_key_exists("Take care of your health", $rows) ) {
   // Then "Get the Most" margin-left: auto
    $layoutclass = 'double-right';
  }
  // If "Understand your benefits and costs" and "Take care of your health"
  if (array_key_exists("Understand your benefits and costs", $rows) && array_key_exists("Take care of your health", $rows) ) {
    // Then "Take Care" margin-left: auto
    $layoutclass = 'double-split';
  }
} 

// If 1
if ($rowCount == 1) {
  // If "Get the most out of your benefits" and "Take care of your health"
  if (array_key_exists("Take care of your health", $rows) ) {
    // Then "Take care of your health" margin-left: auto
    $layoutclass = 'single-right';
  }
  // If "Understand your benefits and costs" and "Take care of your health"
  if (array_key_exists("Get the most out of your benefits", $rows) ) {
    // Then "Get the Most" margin-left: auto, margin-right: auto
    $layoutclass = 'single-center';
  }
}
